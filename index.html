HanzenPHP

is extended for CodeIgniter.

Extend CodeIgniter

How to install

  • 1. copy hanzen_php folder to application/third_party/
  • 2. copy all files inside core folder to application/core
  • 2. open config.php file in 'application/config/'
  • 3. change line 109 to $config['subclass_prefix'] = 'HP_';
  • 4. every create controller use 'extend HP_Controller' not CI_Controller anymore.
  • 5. read user guide.
